Project Background

Data Sourcing and Cleanup

Our group used a Kaggle dataset which contains over 55,000 records of video games and their scores and sales. Once this was loaded into Jupyter notebook the preprocessing and data cleanup started to get the data ready for the machine learning function. This included dropping columns that were unrelated and data that was either empty or contained null values.

View Kaggle Dataset

Development

Once a clean dataset was obtained it was time to start changing the categorical data to numerical for testing by using get dummies. The machine learning algorithm we used for our recommender was K- nearest neighbors. This allows us to pull the nearest neighbors based on the users input and make a recommendation.

Implementation and Analysis

Using the Jaccard Metric option our nearest neighbors algorithm was able to accurately make recommendations to the users input. We also added another option for the user if they wanted to get recommendations based on the console they have as well as a minimum critic score.
